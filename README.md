YouVideo
================
> 이것은 유튜브를 닮은 반응형 웹 프로젝트이다.
> 사용한 파일(모두 templates폴더에 존재한다.) : index.html, style.css, main.js, img.jpg
> python venv, django에서 작성했으나 templates폴더만으로도 확인해 볼 수 있다.

실행 환경 
----------------
* IDE : VSCODE
* Django 3.1.7
* python : 3.9.2

실행 방법
----------------
가상환경 설치
`python -m venv myvenv`
가상환경 실행
`python myvenv/Script/activate`
장고 설치
`pip install django`
